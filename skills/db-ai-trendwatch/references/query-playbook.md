# Query Playbook

Use these query patterns with web search or site search. Replace [company] or [feature].

## Company/product docs
- "[company] AI assistant documentation"
- "[company] release notes AI" 
- "[company] pricing AI" 

## Preview/GA status
- "[company] [feature] public preview"
- "[company] [feature] GA release notes"

## Capability scans
- "[company] NL2SQL" 
- "[company] vector search" 
- "[company] RAG" 
- "[company] model serving" 
- "[company] governance AI" 
- "[company] multimodal search" 
- "[company] full-text search" 
- "[company] hybrid search BM25 vector"
- "[company] graph database" 
- "[company] knowledge graph" 

## Research and analyst signals
- "[topic] site:acm.org" 
- "[topic] site:vldb.org" 
- "[topic] IDC report" 
- "[topic] Gartner report" 

## Customer adoption
- "[company] customer story [feature]"
- "[company] case study AI data platform"
