# Output Templates

## Weekly brief (default)

Title: DB AI Trend Brief (YYYY-MM-DD)

1) Top signals (3-5 bullets)
- Signal, company, capability, status, availability

2) What changed since last update
- New GA features
- New preview features
- Pricing or entitlement changes

3) Competitive position
- Who is ahead / behind by capability
- Differentiators and gaps

4) Risks and unknowns
- Missing evidence
- Preview-only features
- Regional/SKU constraints

5) Watch list (next 2-4 weeks)
- Events, expected releases, doc updates
